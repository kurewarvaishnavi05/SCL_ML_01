# SCT_ML_01

## Task 1 - House Price Prediction

This project implements a Machine Learning model using **Linear Regression** to predict house prices.

### Features Used
- Square Footage
- Number of Bedrooms
- Number of Bathrooms

### Technologies Used
- Python
- Jupyter Notebook
- Pandas
- Scikit-learn
- Matplotlib

### Project Goal
To build a simple regression model that predicts house prices based on basic housing features.
